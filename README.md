# jk-etc-home
Configuration and dotfiles for zsh, vim, tmux, screen and more... for /etc and ~/ (home) usage

# Installation

```shell
git clone https://github.com/jkuetemeier/jk-etc-home-dotfiles.git

./setup.sh -l -v
```
