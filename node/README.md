# Node.js

The `setup.sh` can install some default and required node (npm) modules for you.

## Mac OS X

```
brew install node

# install default npm modules with this setup script
node/setup.sh
```

## Linux

```
# Ubuntu / Mint etc.
sudo apt-get install node

# CentOS, Redhat, Fedora
sudo yum install node

# install default npm modules with this setup script
sudo node/setup.sh
```
